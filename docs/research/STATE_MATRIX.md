# State Matrix

## Evidence labels

- **Source-confirmed** means the state is recorded in the route-specific specification with source screenshot and/or DOM evidence.
- **Local-only deterministic** means the clone intentionally supplies stable fixture/UI behavior where the source was empty, defective, unsafe to exercise, or did not consume an intended input. It is not source evidence.
- **Unverified / blocked** means the state was not forced because doing so would require a network failure, upload, download, synchronization, or operational mutation. It must not be described as observed source behavior.

All search, filter, sort, pagination, disclosure, tab and preview controls in the clone may change rendered local UI only. Every upload, download, refresh, sync, create, save, approve, confirm, send, cancel, archive, delete, permission, status, queue, warehouse, receipt and other operational action is hard-disabled and has no mutating callback, reducer or external request.
